Isabella Edel Museum

Aufgabe:

Gestaltung und Umsetzung der Webseite für das Isabella-Edel-Museum, welches sich damit rühmt, ausschließlich Werke von Frauen auszustellen, welche bislang ohne Anerkennung waren.

Idee:

Aufgrund von Marktbegleitern und der damit erwarteten 'User Experience' erfolgt die Gestaltung ausschließlich schwarz-weiß-grau monochromatisch.

Konzeption:

Die Ausarbeitung der Idee erfolgt mittels der Verwendung des Golden Canon für das Layout mit wenigen Gestaltungselementen und großzügigen Weißflächen.

Umsetzung:

Das erstellte Layout wird mittels HTML und CSS in eine funktionierende Front-End-Webseite umgesetzt. Erwähnenswert ist, dass die Webseite ohne die Verwendung von JavaScript vollständig in Visual Studio Code umgesetzt ist.
